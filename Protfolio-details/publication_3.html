稿件自動化審稿工具

2025 5月

專案簡介

協助一間活動公關公司開發文學獎稿件自動審稿工具，成功取代原先仰賴人工作業與第三方檢測平台的流程。透過 Python 建置 GUI 介面（Tkinter），並整合 Selenium 自動化操作，實現一鍵批次上傳與審核，大幅提升審稿效率。

✅ 作業時間縮短約 80%
✅ 成本降低超過 60%
✅ 減輕團隊審稿負擔
✅ 建立公司內部可複製的自動化流程架構

這套工具讓非技術人員也能輕鬆上手，並支援後續活動的快速部署，成功將繁瑣的流程轉化為穩定高效的操作介面。

Project Overview

Assisted an event PR company in developing an automated manuscript review tool for literary awards, successfully replacing the previous manual process and third-party detection platforms. Built a GUI interface using Python (Tkinter) and integrated Selenium automation to achieve one-click batch upload and review, significantly improving review efficiency.

✅ Process Efficiency +80%
✅ Cost Down > 60%
✅ Labor Load Reduction
✅ Workflow Standardization & Automation

This tool enables non-technical personnel to easily get started and supports rapid deployment for subsequent events, successfully transforming complex processes into a stable and efficient operational interface.

專案簡介

本專案以 React 與 Next.js 建構前端架構，串接後端 n8n 自動化流程，打造一個讓使用者能即時與顧問互動的線上服務平台。 透過 n8n 節點化流程取代繁瑣的後端程式，使資料串接、除錯與維護更直覺，並可靈活擴充後續功能。

✅ 縮短開發間約 40%
✅ 提升後端除錯便利性。
✅ 建立可複製的自動化流程架構

Project Overview

This project was built using React and Next.js for the front-end architecture, integrated with an n8n automation workflow on the back end to create an online consulting platform that enables real-time interaction between users and advisors. By adopting a node-based workflow in n8n to replace complex backend code, the system makes data integration, debugging, and maintenance more intuitive, while allowing flexible scalability for future features.

✅ Reduced development time by approximately 40%
✅ Improved backend debugging convenience
✅ Established a replicable automation framework.
